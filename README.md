🚀 A premium, cinematic, scroll-driven web application designed to showcase mall ecosystems, retail opportunities, and immersive brand storytelling.

# ✨ Overview

**Mall DigiDeck** is not just a website — it’s a **product-level interactive experience**.

Built with a focus on:

* 🍎 Apple-style storytelling UX
* 🎬 Cinematic animations
* 🎮 3D interaction
* ⚡ Smooth performance

# 🔥 Key Features

## 🎬 Video Scroll Sync

* Scroll controls video playback
* Creates immersive storytelling experience

## 🍎 Apple-Style Scroll Story

* Section pinning
* Horizontal scroll animation
* Narrative-driven flow

## 🎮 3D Experience (Three.js)

* Interactive 3D elements
* Smooth rendering with React Three Fiber

## ⚡ GSAP Animation Engine

* ScrollTrigger-based animations
* Smooth transitions and reveals

## 🛍 Retail Showcase

* Animated brand grid
* Interactive hover effects

## 🎤 Events Section

* Timeline-based UI
* Visual storytelling

## 🗺 Interactive Map

* Zone-based exploration
* Expandable structure

## 💰 ROI Calculator

* Business simulation
* Conversion-focused UX

# 🧠 Architecture

> “Separated storytelling (UI sections) from logic to ensure scalability and modular design.”

```bash
src/
├── app/            # Next.js App Router
├── components/     # Reusable UI components
├── sections/       # Story-driven UI sections
├── hooks/          # Custom hooks (GSAP, scroll)
├── data/           # Static structured data
├── utils/          # Helpers
```

# 🚀 Getting Started

## 1️⃣ Clone the repository

```bash
git clone https://github.com/SaloniSsSaini/mall-digideck.git
cd mall-digideck
```

## 2️⃣ Install dependencies

```bash
npm install
```

## 3️⃣ Run locally

```bash
npm run dev
```

👉 Open: http://localhost:3000
